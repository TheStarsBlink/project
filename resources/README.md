# Chrome 浏览器安装说明

本项目需要使用 Chrome 浏览器。请按照以下步骤获取：

1. 访问 Chrome 官方下载页面：
   - 中国用户：https://www.google.cn/chrome/
   - 国际用户：https://www.google.com/chrome/

2. 根据您的操作系统选择合适的版本：
   - Windows 64位
   - Windows 32位
   - Mac
   - Linux

3. 下载并安装 Chrome 浏览器

4. 安装完成后，将 Chrome 浏览器的安装路径配置到项目的相关配置文件中

注意：请确保下载的 Chrome 版本与项目要求的版本兼容。
